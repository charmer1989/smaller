### 安装

可以在组里搞1台开，其他人只要使用它就行了

```
docker pull h2non/imaginary
docker run -p 9000:9000 h2non/imaginary -cors -gzip
npm i -g smaller
```

### 使用

cd到image目录

```
smaller
```
